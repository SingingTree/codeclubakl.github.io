Lots of Boxes

We're using JavaScript to make some clickable boxes.

What skills do I need?

It helps if you know the difference between HTML, CSS and JavaScript.

We recommend doing our Make a Page and Idea Generator projects before this one.

Part 1: Put boxes on Neocities

Download this file to your computer. Right click and choose 'Save Link As…' or 'Download linked file as…'

Download: boxes.html

1. Download the file linked above.

2. Go to neocities.org and log in (or create a new account)

3. In Neocities, go to the dashboard.

4. Upload the boxes.html file from your computer to Neocities.

5. In Neocities, view boxes.html and make sure it works.

6. You should see a page with a big empty square outline, and a button that says "add a box".

Part 2: Save one hundred clicks

To fill the whole outline with boxes, you need to click the button one hundred times!

That sounds hard on our wrists, so let's make it automatic.

Look for the line var start = function () { near the end of the file.

Right now, it adds one box.

Use a loop to make it add 100 boxes, by calling addBox() 100 times.

Hint:

Here's a "for loop":

for (var i = 0; i < 100; i++) {
  alert("I am so cool, number " + i);

This loop repeats the code inside the curly brackets 100 times. (Ask a mentor if you want to know exactly what each part means).

Part 3: Flip button

We don't need the 'add a box' button any more, so let's change it to a flip button that switches all the boxes at once.

We have a list called 'boxes' which contains all 100 boxes. They are numbered, starting from 0, up to 99

Here is a function that flips the first three boxes:

var switchAllBoxes = function () {
  var box = boxes[0]; //get first box from the list.
  switchBox(box);

  var anotherBox = boxes[1]; //get second box.
  switchBox(anotherBox);

  var yetAnotherBox = boxes[2]; //here we go again...
  switchBox(yetAnotherBox);
}

Add this new function after the addBox function.

Functions don't do anything until they are called. You need to call the function to see if it works.

Make it so clicking on the button calls the function switchAllBoxes instead of calling addBox like it used to.

Clue:

You need to change a line near the end of the program, that starts with button.addEventListener. You'll replace the word 'addBox' with 'switchAllBoxes'.

If it works, the button will flip the first three boxes from off to on.

What if it's not working?

If something's not working, it sometimes helps to look at the error messages. Most web browsers let you do this – ask one of our mentors or try to work it out for yourself. The key words you're looking for are "Developer Tools", "Console", or "JavaScript Console".

Error messages appear as red words in the console. Clicking on the error message will show you where the problem is in the JavaScript.

I want to flip ALL the boxes

Right now, our flip button only flips 3 boxes. We want to flip all 100! But we don't want to write 200 lines of code to do it. Let's use a loop.

Remember our loop code from before?

for (var i = 0; i < 100; i++) {
  alert("I am so cool, number " + i);

Use the loop code again, but this time make it call this code 100 times:

var box = boxes[i];

Change the function switchAllBoxes to use a loop. Inside the loop, use the switchBox function and switch all the boxes.

CHECK: Clicking the button makes every box switch from black to yellow, or from yellow to black.

Part 4: Random Button

Add a NEW button to the page.

You'll need to add one line of HTML, to make the button appear on screen.

You'll need to make the new button have an id which is different from the old button. That's so we can make it do something different.

You'll need to add two lines of JavaScript to make the button call a function when clicked (otherwise it does nothing).

See how you go from there.

Here is a new function for the button to call.

var switchSomeBoxes = function () {
  //count from 0 to 99
  for (var i = 0; i < 100; i++) {
    var box = boxes[i]; //get box number 'i'
    if (Math.random() > 0.5) switchBox(box);
  }
}

Try adding this functions, and make the new button call it.

If it works, it will create a random jumble of light and dark squares.

What does this code do?

if (Math.random() > 0.5)

Math.random() picks a random number between zero and 1. It chooses decimal numbers, like 0.31 or 0.833 or anything.

Why did I choose 0.5? What happens if you change it to 0.9? What about 0.1?
