Welcome to Code Club Auckland.

Here's our QBasic tutorial!

QBasic is an old programming language. It's not used much any more, but it used to be the most popular language for beginners.

Old computer magazines used to include programs written in Basic - not on a disk, but just printed in the magazine. You had to type all the code into your computer before you could run the program!

Installing QBasic

Windows: 1. Download the windows QBasic zip 2. Extract the zip file 3. Double-click on qb.bat

OS X: 1. Download the OS X QBasic zip 2. Extract the zip file 3. Double-click on qb.command

If you see a blue blocky screen then it's working!

Your first program

Write this

PRINT "Hello World"

Then press F5 (you might need to press Function + F5 on a laptop).

You should see a black screen that says Hello World.

Asking a question

1 PRINT "What is your name?"
2 INPUT name$
3 PRINT "Welcome, " + name$

Making a choice

1 PRINT "Do you go north or south?"
2 INPUT choice$
3 IF (choice$ = "up") THEN
4  PRINT "I can see forever!"
5 END IF
6 IF (choice$ = "down") THEN
7   PRINT "You are hiding in the ground."
8 END IF

Jumping around

The computer goes through your program, line by line, as if it was a list of instructions.

1 PRINT "First print this"
2 PRINT "Then print this"

What if we don't want to go line by line?

1 PRINT "What's the password?"
2 INPUT pass$
3 IF (pass$ = "secret") THEN
5    GOTO 10
6 END IF
7 PRINT "That's not the password."
8 GOTO 1
10 PRINT "You win!"

Line numbers

Line numbers are hard to remember so we can use labels instead. This makes your program easier to understand.

This program works exactly like the program above, but it uses a label called 'outside' instead of line numbers.
