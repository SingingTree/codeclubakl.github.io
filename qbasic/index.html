QBasic Adventure

QBasic is a programming language from the 80s. It's not used much any more, but it used to a very popular language for beginners.

There was no internet back then. Some magazines would have the code for programs written out. You had to type all the code into your computer before you could run the program!

It was pretty weird.

Installing on Windows

1. Download the windows QBasic zip
2. Extract the zip file
3. Double-click on qb.bat

Installing on OS X

1. Download the OS X QBasic zip
2. Extract the zip file
3. Right-click on qb.command and choose 'open'

Your Mac might say you cannot run the program because it's from an unidentified developer.

You can still run it by right-clicking or holding control while you click on it.

Choose 'open' from the menu.

Choose 'open' again.

Installing on Linux

We haven't prepared an easy Linux download.

You'll have to install DOSBox, download QBasic 4.5, then run DOSBox from the terminal giving it the path to qb.exe.

The QBasic Editor

If it's working you will see this:

Try it out

Write this

CLS
PRINT "Hello!"

Then press F5 (you might need to press Function + F5 on a laptop).

You should see a black screen that says Hello! and then Press any key to continue.

Press a key to go back to the editor.

CLS means 'clear the screen.'

PRINT writes letters onto the screen.

Making an Adventure Game

We're going to make a text game where the player moves between places by choosing where to go next.

We need to:

  • Print words to the screen.
  • Ask the player to type in their choice.
  • Do something different based on their choice.

Step 1: Asking a question

CLS
PRINT "Should we go north or south?"
INPUT mychoice$

Run this program (F5).

The computer runs the program line-by-line.

At the INPUT line it stops, and waits for you to type in something and press ENTER.

It saves your answer into a variable called mychoice$. A variable is like a box that you can put something inside.

Step 2: React to the user's choice

This is the same code as above, with one line added.

CLS
PRINT "Should we go north or south?"
INPUT mychoice$
IF (mychoice$ = "north") THEN PRINT "You found a secret cave!"

Try running this program. What happens if you type 'north' [ENTER] as your answer?

What happens if you write 'North' with a capital letter instead?

Step 3: Jumping around

The computer starts from the first line goes through the program, line by line.

We can use GOTO to make the computer jump to a different line instead.

In this example, start: is a label. It doesn't do anything by itself, but you can jump to it. cave: is also a label.

CLS
start:
PRINT "Should we go north or south?"
INPUT mychoice$
IF (mychoice$ = "north") THEN GOTO cave
PRINT "You can't go that way. Please try again."
GOTO start

cave:
COLOR 15
PRINT "You found a secret cave!"
END

Can you run this program in your head?

Can you add some code so that typing in 'south' makes you go to another location?

Try making something! Remember you can always ask for help.

Extra Ideas

These are only if you want an extra challenge. Look at these examples and see if you can use these ideas in your project.

PLAY "c d e f g"

coins = 4
PRINT coins
PRINT "coins"
PRINT "You have " + coins + " dollars"

coins = 3
IF coins >= 2 THEN
  PRINT "You buy a hat."
  coins = coins - 2
  PRINT "You have $" + coins + " left."
ELSE
  PRINT "You can't afford a hat right now."
END IF
