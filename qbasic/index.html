QBasic Project

QBasic is an old programming language. It's not used much any more, but it used to a very popular language for beginners.

Old computer magazines used to include programs written in Basic - not on a disk, but just printed in the magazine. You had to type all the code into your computer before you could run the program!

Installing QBasic

Windows:

1. Download the windows QBasic zip
2. Extract the zip file
3. Double-click on qb.bat

OS X:

1. Download the OS X QBasic zip
2. Extract the zip file
3. Double-click on qb.command

Your Mac might say you cannot run the program because it's from an unidentified developer.

You can still run it by right-clicking or holding control while you click on it.

Choose 'open' from the menu.

Choose 'open' again.

Linux

We haven't prepared an easy Linux download. You'll have to install DOSBox, download QBasic 4.5, then run DOSBox from the terminal giving it the path to qb.exe.

Making an Adventure Game

We're going to make a text game where the player moves from room to room by choosing where to go next.

We need to:

  • Print words to the screen.
  • Ask the player to type in their choice.
  • Do something different based on their choice.

Asking a question

CLS
PRINT "Should we go north or south?"
INPUT mychoice$

Run this program (F5).

The computer runs the program line-by-line.

At the INPUT line it stops, and waits for you to type in something and press ENTER.

It saves your answer into a variable called mychoice$. A variable is like a box that you can put something inside.

Using the answer to do something

This is the same code as above, with three lines added.

CLS
PRINT "Should we go north or south?"
INPUT mychoice$
IF (mychoice$ = "north") THEN PRINT "You walk to Birkenhead."
IF (mychoice$ = "south") THEN PRINT "You walk to Epsom."
PRINT "The end."

Try running this program and entering 'north' or 'south' as your answer.

What happens if you write something else?

What happens if you write 'North' with a capital letter at the start?

Jumping around

The computer starts from the first line and does each instruction in order.

We can tell the computer to jump to a different line using a GOTO command.

We add labels for the computer to go to. A label is a word with a colin : at the end.

CLS
home:
PRINT "Should we go north or south?"
INPUT mychoice$
IF (mychoice$ = "north") THEN GOTO birken
IF (mychoice$ = "south") THEN GOTO epsom
PRINT "I don't understand. Please try again."
GOTO home

birken:
PRINT "You swim to Birkenhead."
END

epsom:
PRINT "You fly to Epsom."
END

Extra for Experts

Expand this for some extra ideas:

Take a look at this.


    coins = 7
    PRINT coins
    PRINT "coins"
    PRINT coins + "coins"

What's the difference between coins and "coins"?

Can you run this code in your brain? See if the computer gets the same result.


    coins = 7
    PRINT "Do you buy an apple?"
    INPUT a$
    IF (a$ = "yes") THEN coins = coins - 2
    PRINT "You have " + coins + " coins left."

coins = coins - 2 is a strange thing to say! It looks impossible.

What is means is, the NEW value for coins should be the OLD value of coins minus 2.

coins = coins - 2
coins = 7 - 2
coins = 5

IF and END IF


    coins = 7
    PRINT "Do you buy an apple?"
    INPUT a$
    IF (a$ = "yes") THEN
        coins = coins - 2
        PRINT "You bought an apple."
    END IF
    PRINT "You have " + coins + " coins left."

Something special happens when THEN is the last word on a line.

END IF is a strange thing to say. What does it mean?
