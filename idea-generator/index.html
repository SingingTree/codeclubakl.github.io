Idea Generator

We're using JavaScript to make a TV Series Idea Generator.

It will help you come up with great ideas for a TV series, like this:

A dance instructor and a rebellious teenager solve mysteries on a tropical island.

What skills do I need?

This activity is for people who've already done some basic HTML and CSS.

If you've never written HTML, please do this activity instead: Make a Page

Part 1: Starting from something

Sometimes we start from nothing, and write all our own code.

But most of the time, we save time by using other people's code as a starting point.

Here is a starting point for you: TV Concept Generator

1. Open the TV Concept Generator that is linked above.

2. The blue area is the TV Concept Generator.

3. Click the 'Next idea' button. Try it lots of times.

4. Can you explain what the happens when you click 'Next idea'?

The other parts of the page let you edit the app.

This JSFiddle webpage has 3 white boxes.

1. Work out which box shows the JavaScript.

2. In that box, find line number 22

3. For now, we are only looking from line 22 to the end.

4. What's the connection between this code and what happens when you click 'Next idea'?

Cool, now we're ready to make some changes.

Part 2: pushing the limits

1. Add two new people ideas to the app.

2. Add two new job ideas to the app.

Clues:

This is the symbol for an empty list: []

This is the command to add something to a list: push

Remember, you only need to look from line 22 to the end.

Part 3: Teamwork

I like shows that have two main characters.

Make it so when you click the button, it picks two people from the list.

You should include the words "and a" between the two people.

When it's working it should make ideas like this:

A school teacher and a time traveller who runs a hotel

A school teacher and a school teacher who solves mysteries

Clue:

Look for the word myConcept in 2 lines of code near the end. Take a close look at those two lines, and what happens when you click 'Next idea'.

Part 4: New York City

We need to say what city the show is set in.

It should say in New York or in London at the end of each concept.

1. We will need a list of cities

2. We need to pick a random city, and display it somehow.

Part 5: Make it pretty

Take a break from the JavaScript and improve the CSS.

Do you remember CSS from our HTML and CSS activity?

Use CSS to style each part of the app – the button, the h1, and the h2.

Part 6 – Move to Neocities

We are working on a website called JSFiddle. JSFiddle is only for coders like us. If you want to show your app to other people, you don't want to have all the code right up there!

Let's put the TV Show Idea Generator on your Neocities page instead.

You will need to download this file to your computer. Right click and choose 'Download As...' or 'Save Target As...'

  • tvshow.html

1. Download the file linked above.

2. Go to neocities.org and log in (or create a new account)

3. Upload the tvshow.html file from your computer.

4. View the tvshow.html and make sure it works.

Now you have a TV Show Idea Generator on your own website. But it doesn't have the changes you made.

1. Edit tvshow.html in Neocities.

2. Make the same improvements that you made on JSFiddle, so it looks better (CSS) and generates better ideas (JavaScript).

3. Once you've finished, view tvshow.html again and check that it's working.

And we're done.
