Webpages #2

Hi, if you're here you should have a neocities account, and 3 tabs open in your browser: one to edit html, one to edit css, and one to view your website.

Tip: Which is which?

HTML has a lot of pointy brackets like these: < >

CSS has a lot of curly brackets like these: { }

Background Picture

We use CSS to change the background picture. Open the tab that is editing style.css (it's the one with curly brackets.)

Add this code to the end of the file, starting on a new line.


body {
    background-image: url('http://mgatland.neocities.org/countryside.jpg');
    background-size: cover;
    background-attachment: fixed;
}

This will give your page a background picture! Remember to save and refresh first.

Tip: Click 'Save Changes', then switch to the tab that shows your website, then press Refresh.
