Hi, if you're here you should have a neocities account, and 3 tabs open in your browser: one to edit html, one to edit css, and one to view your website.

Tip: Which is which?

HTML has a lot of pointy brackets like these: < >

CSS has a lot of curly brackets like these: { }

Changing Colours Colors

We can change the color of anything using CSS. Open the tab that is editing style.css (it's the one with curly brackets.)

Look at your CSS file. Can you see where the colors are set? Try changing them from 'red' and 'blue' to something else.

When we're programming, we have to spell in US English, not NZ English. That means we spell 'color' without the 'u'.

Luckily, only a few words are different.

Did you change the colors? Remember to save and refresh.

Your CSS code might look like this now:


body {
  margin: 0;
  color: purple;
  background-color: orange;
}

Change the heading

Let's add a new CSS rule to make the page heading look different.

Add this to the end of your CSS file, keeping what is already there.


h1 {
  text-align: center;
  font-size: 90px;
  color: white;
}

Save your changes, switch tabs to your page and check what happened.

You should see that the heading has changed.

About mistakes

When you are coding you will make a LOT of mistakes. Even experts make a lot of mistakes. Just missing one '>' symbol can make your page display strangely.

If you're at a Code Club, the mentors can help you spot mistakes. If you're by yourself, you might have to carefully compare your code with the examples. Look for tiny differences like one symbol being moved or missing.

Editing your HTML

Switch tabs to where you're editing your HTML file. (It has lots of pointy brackets like these: < and >)

For now, we only want to change what's "inside the body of the page".

"inside the body" means the text between <body> and </body>. Those are the markers for the start and end of the page body.

Your page body probably looks like this:


  <body>
    <main>
        <h1>Welcome to my page</h1>
        <p>This is a paragraph.</p>
    </main>
  </body>

h1 is short for 'heading 1'. It is the biggest heading.

p is short for 'paragraph'

Try adding some more paragraphs and changing the text.

Adding an image.

Try adding this. Remember to put it somewhere inside the body.


  <img src="http://mgatland.neocities.org/countryside.jpg">
>

Does it work? You can change it to a different image by changing the link inside the " quotation marks ". Try a Google Image Search.

When you find an image on Google, make sure you click 'View Image' until you have a page that only shows your image, no Google stuff around the edge. Then copy the URL from the address bar.

So you're editing your HTML. You can change what the page says, and add images, but you can't change colors or add borders.

HTML is for the content, but not the layout.

How CSS works

A CSS rule controls how your HTML is displayed.

First we say which part of the HTML the rule is for. This could be body for the whole page, or h1 for just the heading.

Then we use a curly bracket { to say 'the instructions start now.'

an instruction looks like this:


  color: white;

Note it has a colon in the middle and a semi-colin at the end.

At the end, you need to say you've finished by putting a closing curly bracket }

So here's an example of a complete CSS rule:


img {
  border: 10px dashed green;
  margin: 30px;
}

Can you tell what it will change?
