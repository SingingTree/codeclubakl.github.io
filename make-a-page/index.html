Webpages

We're going to make a web page on the internet.

You can make a page about something that you're interested in. First we have to get set up.

Here are the steps we're going to do:

  • Make a Neocities account
  • Set up a CSS file
  • Open 2 tabs for editing our site and 1 tab for viewing our site
  • Edit the HTML file
  • Now you're ready to make your own site

Make a Neocities account

  • Go to neocities.org
  • Under 'Get Started Now', type in a name for your site and click 'Get Started'
  • Fill out the form to create your account. We are using the free option.
  • Once you finish you will see a page that says 'My Website' at the top center.

Set up Neocities with HTML and CSS

  • Right-click on this link and save the file style.css to your computer.
  • In Neocities, click Upload and upload that same file (style.css)
  • In Neocities, delete cat.png and not_found.html
  • You should have 2 files left: index.html and style.css.

Editing

I like to work with three tabs open.

Tab 1: Open your website in a new tab by right-clicking on the big red link under 'My Website'.

This tab is where you will look at your page.

Tab 2: Hold your mouse over index.html and the word 'edit' appears. Right click and open in a new tab.

This tab is where you edit your page.

Tab 3: Edit style.css the same way you did for index.html.

You should have 3 tabs: one to view your site, one to edit index.html and one to edit style.css.

Edit your HTML page

  • Go to the tab for index.html
  • Delete everything until it's empty.
  • Then paste this in instead:

<html>
  <head>
    <link rel="stylesheet" type="text/css" href="/style.css">
  </head>
  <body>
    <main>
        <h1>Welcome to my page</h1>
        <p>This is a paragraph.</p>
    </main>
  </body>
</html>

Click 'save changes' to save. (You might have to scroll down. It's the big red button in the middle.)

Now switch to the tab that shows your website. Refresh the pages and you should see your changes appear.
