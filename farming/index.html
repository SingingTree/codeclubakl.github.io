Farming Game

We're using JavaScript to make a farming game.

In this game, you plant trees, wait for them to grow, then harvest them for money.

You use the money to buy more trees! So you can make more money! And buy even more trees! IT GOES ON FOREVER

What skills do I need?

It helps if you know the difference between HTML, CSS and JavaScript.

We recommend doing our Make a Page and Idea Generator projects before this one.

Part 1: Start a farm on Neocities

Download this file to your computer. Right click and choose 'Save Link As…' or 'Download linked file as…'

Download: farming.html

1. Download the file linked above.

2. Go to neocities.org and log in (or create a new account)

3. In Neocities, go to the dashboard.

4. Upload the farming.html file from your computer to Neocities.

5. In Neocities, view farming.html and make sure it works.

6. You should see a page with a black rectangle. It will turn brown in 2 seconds, then green 2 seconds later.

7. Once the rectangle is green, click on it. It will turn black again, and you'll get some money.

8. The button won't work yet.

Now you have your own copy of the farming game, so you can start improving it.

From the Neocities dashboard, edit farming.html.

I recommend having one browser tab for editing, and one for viewing the page.

Remember, each time you make a change you need to save (on the editing page) and then refresh (on the viewing page)

Cool, now we're ready to make some changes.

Part 2: Draw some plants

That black rectangle is supposed to be a plant. Let's draw some pictures of the plant's life cycle:

  1. A small plant
  2. A medium-sized plant
  3. A large plant, with delicious apples on it, so the player knows it is ready to harvest.

You can use any drawing tool you like. I suggest gifpaint.com.

In gifpaint, click 'New GIF' up the top. Set the width to 64, height to 128, and frames to 1.

When you've finished drawing, click 'Encode GIF' (up the top) then click Download to download the gif file to your computer.

On your computer, rename to files to something simple like "small.gif", "medium.gif" and "large.gif".

Then upload the files to Neocities.

Part 3: Put the plants in the game

We can use CSS to make the plants show up in-game.

Go back to editing farming.html. Look for the CSS – it's near the top.

There are some CSS rules called '.apple-tree.small', '.apple-tree.medium' and '.apple-tree.large' which control what the plants look like.

Right now they set the background-color. You can use an image instead by using a CSS command like this:

background-image: url("/small.gif");

Get your new plant images showing up in the game.

Part 4: Spending

Time to spend all that hard-earned money.

Look for a line that says var buyPlant = function() {, near the bottom of the file.

This function controls what happens when the button is clicked.

Make the "buy an apple tree" button work.

It should check if you have enough money for a plant.

If you do, it should subtract that money from your balance, then add a new plant.

If you don't, it should pop up a message saying "You don't have enough money."

Clues:

>= is a symbol that means "greater than, or equal to." It's useful for price checks.

if (age >= 5) {
  alert("You are old enough to go to school.");
  alert("Have fun!");
} else {
  alert("No school for you.");
}

money is the variable that holds how much money you have.

If you add or subtract from money, you need to call updateMoneyDisplay() to make the number on screen change. It's not automatic.

money = money + 10; //payday!
updateMoneyDisplay(); //show my new balance

addPlant("apple-tree") is a function I made that will add a new plant to the farm.

Good luck with your farm!
