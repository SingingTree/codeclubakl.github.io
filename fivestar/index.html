Five Star

We're going to make stars appear and disappear on our page.

It could be for giving ratings, or it could be for a game where you click on stars to get stardust.

(This activity uses JavaScript and jQuery.)

What skills do I need?

It helps if you know the difference between HTML, CSS and JavaScript.

We recommend doing this after doing our "Make a page" and "Idea generator" activities first.

Part 1: Set up Neocities

Log in to Neocities.org. (If you don't have an account, create one. It's free.)

Make a new page:

Call it fivestar

Open a tab to edit the new page.

Open a tab to view the new page.

Delete all the default code

Copy and Paste this code instead:

<!DOCTYPE html>
<html>
  <head>
    <meta charset='utf-8'>
    <script src="//ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script>
    <title>Five Star</title>
    <style>
    /*This is the start of the CSS*/

    .star-box {
      width: 600px;
      height: 120px;
      background-color: grey;
    }

    .star {
      width: 120px;
      height: 120px;
      background-image: url("http://codeclubakl.github.io/linked/goldstar.png");
      float: left;
    }

    /*end of CSS */
    </style>
  </head>
  <body>
    <h1 class="title">Five Star Rating</h1>
    <div class="star-box">
      <div class="star"></div>
    </div>
    <button class="button1">hello</button>
    <script>
//Hello! This is the start of the JavaScript.

var greeting = function () {
  alert("Hello there!");
}

var addStar = function () {
  var star = $('<div class="star"></div>');
  $(".star-box").append(star);
}

var removeStar = function () {
  $('.star').first().remove();
}

$(".button1").on("click", greeting);

//This is the end of the JavaScript.
    </script>
  </body>
</html>

CHECK that you copied all the code. Save the page, then view it.

What is this?

You will see a page with one star and one button.

Clicking the button makes a message pop up.

Part 2: Make the button add a star

We use JavaScript to make the button call a function when you click on it.

1. Find the JavaScript. It starts with <script> and ends with </script>

2. Find this line: $(".button1").on("click", greeting);.

3. Replace the word greeting with addStar.

We just changed what happens when you click on the button, using JavaScript.

CHECK: clicking the button now adds a star.

Part 3: Fix the button's label

The button's label says "hello". It should say something like "Add a star".

1. Find the HTML that adds a button to the page.

This is NOT the same line we were before! It's not part of the JavaScript.

2. Give the button a better label.

CHECK: The button has a better label instead of "hello".

Part 4: Remove stars

Let's make a new button that removes stars.

1. Add a new button to the HTML. Put it next to the other button, and give it a different class.

2. In the JavaScript, make your new button call the removeStar function when it is clicked.

When we say $('.something'), it tells the computer to look in the HTML for class="something". The little dot at the start means "this is a class".

CHECK: You have 2 buttons. One adds stars, one removes stars.

Part 5: Buying stars

Let's make this a game. You need "stardust" to buy new stars.

At the start of the JavaScript, make a variable to keep track of our stardust.

(Do you remember where the JavaScript starts?)

var starDust = 50;

We also need a way to show the player. Let's add some HTML just after the buttons:

<h2 class="dust-display">star dust display goes here.</h2>

Next, let's make buying stars cost 10 stardust. You will need to add this code somewhere inside the addStar function.

starDust = starDust - 10; //costs 10
$('.dust-display').text("You have " + starDust + " star dust");//show my new balance

CHECK: When you add a star, your star dust decreases.

CHECK: What happens when you run out of star dust?

Part 4: No overdraft

You should not be allowed to buy stars unless you have enough star dust.

You will need to change the addStar() function to make this work.

Here are some clues:

An age check program:

var age = 14;
if (age >= 5) {
  alert("You are old enough to go to school. Have fun!");
} else {
  alert("No school for you.");
}

>= means "greater than, or equal to." It's useful for price checks.

CHECK: If you have 10 or more stardust, you can buy a star.

CHECK: Otherwise, you can NOT buy a star. No new star appears.

Extra if we have time

What if clicking on a star gave you some stardust?

Maybe you could do it. Try changing the addStar function, by adding this to the end of it:

star.on("click", greeting);

Check that, and see what clicking on stars does.

...now instead of making it run the "greeting" function, you can make it run a different function... maybe even make your own.
